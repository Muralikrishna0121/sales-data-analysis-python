# Sales Data Analysis Using Python

## Project Overview
This project analyzes sales data to identify revenue trends, top-performing products, customer behavior, and regional performance.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. KPI Calculation
5. Data Visualization
6. Business Insights

## Key KPIs
- Total Sales
- Average Sales
- Maximum Sale
- Minimum Sale

## Visualizations
- Product Line Sales
- Country-wise Sales
- Monthly Sales Trend
- Sales by Year
- Deal Size Distribution
- Top Customers
- Order Status Analysis

## Business Insights
- Identified top-performing product categories.
- Analyzed sales trends across countries and years.
- Evaluated customer contribution to revenue.
- Generated actionable business insights from sales data.
